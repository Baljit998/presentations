## arrays are faster than objects
instead of storing lots of data in objects.. 
							
	const points = [{
		x: 1,
		y: 4 
	},{
		x: 2,
		y: 5
	}, ..., {
		x: 99999,
		y: 99999
	}]
store it in arrays 
								
	const x = [1, 2, ..., 99999];
	const y = [4, 5, ..., 99999];
six times faster ``` Starting testcase: "Save as objects" Testcase finished: x 6.53 ops/sec ±17.67% (21 runs sampled) ----------------------------------- Starting testcase: "Save as arrays" Testcase finished: x 35.68 ops/sec ±1.67% (43 runs sampled) ```
## preallocate large arrays

instead of pushing into an array
of undefined size...


	const gridWidth = 1000000;

	const slowGrid = [];

	for(let i = 0; i < gridWidth; i++){
		showGrid.push(random(0, 1));
	}

...create large arrays with new Array()
and assign by index



	const grid = new Array(gridWidth);

	for(let i = 0; i < gridWidth; i++){
		grid[i] = random(0, 1);
	}
twice as fast ``` Starting testcase: "Push into array" Testcase finished: push x 30.71 ops/sec ±3.72% (42 runs sampled) ----------------------------------- Starting testcase: "Assign into preallocated array" Testcase finished: assign x 72.57 ops/sec ±2.55% (50 runs sampled) ```
